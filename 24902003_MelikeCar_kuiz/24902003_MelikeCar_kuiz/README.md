## Proje Açıklaması

Bu proje, Web Programlama I dersi Quiz Projesi kapsamında geliştirilmiş tek sayfalık (one-page) kişisel portfolyo / CV web sitesidir. Soft pembe tonlarda özgün bir tasarıma sahiptir.

### Özellikler

- ✅ Sabit (fixed) navigasyon + smooth scroll
- ✅ Profil fotoğraflı Hakkımda bölümü
- ✅ Tablo ile gösterilen eğitim bilgileri
- ✅ Animasyonlu beceri çubukları (IntersectionObserver)
- ✅ Proje kartları (hover efektli)
- ✅ mailto destekli iletişim formu
- ✅ GitHub, LinkedIn ve e-posta linkleri
- ✅ Responsive (mobil uyumlu) tasarım
- ✅ CSS değişkenleri (:root) ile renk paleti
- ✅ Flexbox + CSS Grid düzeni
